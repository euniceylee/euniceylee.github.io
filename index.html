Favorite Projects

The Wrong Man

NBC News

Lead designer and developer of a story that hit more than 450,000 page views (2nd most viewed NBC specials story at the time). Collaborated with editorial and photo teams to decide which visuals would tell the story in the most meaningful ways. Used HTML, CSS, JavaScript, Node, Archie mark-up language.

Mobile Homepage

NBC News

Created design comps of the mobile home page so that the main cover story and related stories flow seamlessly together. Transformed a desktop design into a responsive, mobile friendly version that was not only aesthetically pleasing but also practical and efficient.

Cartograms

NBC News

Designed cartograms to be geographically accurate and, at the same time, familiar to the eye. Conducted in-depth research on existing cartograms, then sketched and prototyped how the map would look and work. They helped visualize how battle states voted in past elections and where Team USA Rio Olympians hail from.

Olympic Team Refugees

NBC News

Created graphics using D3.js and Illustrator to depict how far Team Refugee athletes traveled before finding refuge. See 18-year-old Syrian swimmer Yusra Mardini's journey here.

What Actually Treats Your Cold?

Class Project

I designed, developed, and wrote a service-orientated piece about OTC cold medications. The crux of the project was creating filtering "cards" that explained common active ingredients in drug labels and their uses. Used HTML, CSS, JavaScript, and jQuery.

A Year of Terror in Iraq

Personal Project

I wanted to make a visual representation of ISIS attacks in Iraq this year. The design was inspired by NYT graphics. Used D3.js to create the basemap, converted it to an SVG, and used Illustrator to label details.

Wrecked

Class Project

I created two interactive data viz maps using D3.js, Waypoints, and Leaflet, depicting the path of two shipwreck hypotheses in the Great Lakes. I worked extensively on the JavaScript and D3.js.

StorylineJS

Knight Lab

A storytelling line chart tool. To start off, I conducted research on existing charting tools and stories that use line charts. Next, I worked on sketching and prototyping how the tool would look and work. I created paper prototypes and iteratively tested them. Finally, I worked on a web prototype using HTML, CSS, JavaScript and D3.js. This project is ongoing.
